利用規約

本規約は、Klyph が提供する Web フォント生成・配信機能の利用条件を定めるものです。

このページは Klyph を利用する運営者向けのひな形です。実際の公開条件は各デプロイの運営者が定めてください。

適用範囲

本規約は、本サービスの Web ページ、SDK、API および生成されたフォントファイルを利用するすべての方に適用されます。本サービスを利用した時点で、本規約に同意したものとみなします。

提供内容

本サービスは、登録済みのオープンライセンスフォントから、指定された文字集合に対応する WOFF2 サブセットを生成し、固定 URL で配信します。提供対象、フォント、仕様、上限値は運用上の必要に応じて変更されることがあります。

アカウント作成、ユーザーによる任意フォントのアップロード、生成物の永続保管保証は提供していません。

利用条件

  • 運営者が許可したサイトの表示目的で利用すること。
  • SDK および API の技術的制限、CORS、レート制限、キャッシュ方針に従うこと。
  • 生成されたフォントに適用される SIL Open Font License 1.1 を遵守すること。
  • 障害や仕様変更に備え、利用側で適切なフォールバックフォントを設定すること。

禁止事項

以下の行為を禁止します。

  • 大量の異なる文字集合を送信して、演算資源、待機列、キャッシュまたは帯域を意図的に消費する行為。
  • Origin 制限、WAF、レート制限その他の技術的保護を回避する行為。
  • 違法、有害、権利侵害またはサービス運営を妨害する内容の処理。
  • 脆弱性調査を装った無断スキャン、侵入、リバースエンジニアリングまたはサービス妨害。
  • 本サービスまたは第三者の信用を害する態様での再提供、転売、なりすまし。

知的財産権

サービスのプログラム、画面設計、文章およびブランド表示に関する権利は、運営者または正当な権利者に帰属します。Zen Kaku Gothic New および Zen Maru Gothic とその生成サブセットには SIL Open Font License 1.1 が適用されます。詳細は ライセンス・クレジットをご確認ください。

変更・停止

保守、障害、セキュリティ対応、法令上の必要、上流サービスの変更その他の理由により、事前の通知なく本サービスの全部または一部を変更・制限・停止することがあります。重大な変更は、可能な範囲で本サイトまたは関連サイトに掲示します。

保証および免責

本サービスは現状有姿で提供され、可用性、完全性、特定目的適合性、継続性、生成物の恒久的な保持を保証しません。運営者の故意または重大な過失がある場合を除き、本サービスの利用または利用不能から生じた間接的・付随的損害について責任を負いません。

規約の変更

必要に応じて本規約を変更できます。変更後の規約は本ページに掲載した時点から効力を生じます。重要な変更では、ページ上で更新日と概要を明示します。

お問い合わせ

Klyph 本体に関するご連絡は GitHub Issues をご利用ください。個別デプロイの運用については、その運営者へお問い合わせください。
