構成の根拠

pyproject.toml は直接依存と許容バージョン範囲、requirements.lock は本番イメージの固定バージョン、Dockerfile は実際の Python ランタイムと起動方法を定義します。表示内容とファイルが異なる場合は、リポジトリ内の構成ファイルを正とします。
